HTML é uma Linguagem de Programação?

Voltar ao Sumário

Não, o HTML é uma linguagem de marcação, ou seja, uma linguagem que serve para marcar o conteúdo de uma página web.

O que é uma Linguagem de Marcação

Voltar ao Sumário

Uma linguagem de marcação é uma linguagem que serve para marcar o conteúdo de um documento, ou seja, para indicar ao computador como o conteúdo deve ser apresentado.

CSS é uma Linguagem de Programação?

Voltar ao Sumário

Não, CSS é uma linguagem de estilização, ou seja, uma linguagem que serve para estilizar o conteúdo de uma página web.

Linguagem de Estilização

Voltar ao Sumário

Além de marcar o conteúdo de uma página web, o HTML também pode ser usado para estilizar o conteúdo o recomendado é editar o FRONT-END de seu de seu site com o CSS (Cascading Style Sheets) para manter a organização no código e assim ter a página do formato que desejar.
